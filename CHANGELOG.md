## [1.0.0] - 2025-06-19

### Agregado
- ✨ Sistema de tickets completo con Laravel 11 + Inertia.js + React
- 🎨 Tema oscuro minimalista con fondo negro y tarjetas grises
- 🔐 Sistema de autenticación completo (login, registro, recuperación)
- 📱 Diseño completamente responsivo
- 🇪🇸 Interfaz completamente en español
- 🏢 Branding personalizado de TES LTDA
- 🖼️ Logo personalizado y favicon
- ⚡ Componentes modernos con TypeScript

### Modificado
- 🔄 Redirección automática de página principal a login
- 🎯 Layouts personalizados para páginas de autenticación
- 📏 Iconos de mayor tamaño en páginas de auth (96px/128px)
- 🧹 Eliminación de elementos innecesarios del navbar/sidebar
